# 📊 Adlytix AI — AI Marketing Analytics Dashboard

> Stop guessing. Start knowing.

Upload your ad performance data (Google Ads, Meta, TikTok) and get 
instant metrics + AI-powered insights in seconds.

## ✨ Features

- 📁 Upload CSV or Excel ad exports from any platform
- 📐 Auto-calculates ROI, ROAS, CPC, CTR
- 📊 Interactive charts (ROAS, Spend vs Revenue, CTR, CPC)
- 🤖 AI-generated plain-English insights powered by Claude
- 🏆 Best and worst campaign detection
- 💡 Zero setup required

## 🚀 Live Demo

👉 [adlytix-ai.streamlit.app](https://adlytix-ai.streamlit.app)

## 🧰 Built With

- Python 3
- Streamlit
- Pandas
- Plotly
- Anthropic Claude API

## 📄 How to Run Locally

```bash
git clone https://github.com/YOUR_USERNAME/adlytix-ai.git
cd adlytix-ai
pip install -r requirements.txt
streamlit run app.py
```

Add your `ANTHROPIC_API_KEY` to a `.env` file.
